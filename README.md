# winspace

Windows C-drive cleanup and relocator. Scans your C: drive for large
directories, moves the safe ones to another drive, and leaves NTFS
junctions behind so applications keep working transparently.

> **Status:** pre-alpha. Not ready for general use. See [`plan.md`](plan.md).

A full README — install instructions, usage, risk disclosures, and FAQ —
lands as part of Phase 8 (Task 37) before the `v0.1.0` tag.

See [`spec.md`](spec.md) for design and [`plan.md`](plan.md) for the
implementation roadmap.
